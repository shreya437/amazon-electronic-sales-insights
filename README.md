# Amazon Electronics Sales Insights | Power BI

1 Project Overview
This project analyzes Amazon electronics sales data using Power BI to uncover key business insights related to sales performance, product trends, discounts, and customer ratings. The dashboard is designed to support data-driven decision-making through clear and interactive visualizations.

2 Objectives
- Analyze overall sales and revenue performance  
- Identify top-performing and low-performing electronics products  
- Understand the impact of discounts on sales  
- Examine customer behavior using ratings and review counts  

3 Tools Used
- Power BI  
- Excel / CSV Dataset  

4 Key Insights
- Category-wise and product-wise sales analysis  
- Relationship between discounts and sales volume  
- Performance comparison of highly rated products  
- Identification of best-selling products  

5 Dashboard Features
- KPI cards for quick performance overview  
- Bar and column charts for category comparison  
- Product-level tables for detailed analysis  
- Interactive filters and slicers  

6 File Information
- Power BI template file (`.pbit`) included  
- Dataset is not embedded and can be loaded separately  

7 How to Use
1. Download the `.pbit` file  
2. Open it in Power BI Desktop  
3. Load the dataset when prompted  
4. Explore the interactive dashboard  

8 Use Case
Suitable for Data Analyst, Business Analyst, and Power BI portfolio projects.
